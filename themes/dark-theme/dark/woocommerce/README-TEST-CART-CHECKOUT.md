# اختبار السلة والدفع (dark-theme)

## متطلبات الاختبار
1. **تفعيل ثيم Dark Theme Store**: من لوحة التحكم → المظهر → القوالب → تفعيل "Dark Theme Store".
2. **صفحة السلة**: تأكد أن صفحة السلة (slug: `cart`) معروضة وتستخدم قالب الثيم.
3. **صفحة الدفع**: WooCommerce يجب أن يكون موجّهًا لصفحة الدفع (slug: `payment`) أو استخدام `/payment` كصفحة إتمام الطلب.

## خطوات الاختبار

### الويب (Desktop)
1. أضف منتجًا إلى السلة (من صفحة المنتج أو المتجر).
2. افتح `/cart` وتحقق من:
   - تخطيط عمودين: قائمة المنتجات + ملخص الطلب.
   - شكل البطاقات (حدود، زوايا، ألوان الثيم).
   - أزرار الكمية، الحذف، تحديث السلة، ومتابعة للدفع.
3. اضغط "متابعة للدفع" وافتح صفحة الدفع.
4. تحقق من:
   - تخطيط عمودين: بيانات العميل + مراجعة الطلب.
   - تنسيق الحقول والأزرار وزر "تسجيل الطلب".

### الجوال (عرض ≤820px)
1. غيّر حجم النافذة أو استخدم أدوات المطور (Device toolbar).
2. كرر نفس الخطوات أعلاه وتحقق من:
   - السلة: عمود واحد، عناصر السلة متجاوبة.
   - الدفع: عمود واحد، المراجعة تحت النموذج.

## ملف التنسيق
التصميم الافتراضي لـ WooCommerce يُستبدل عبر:
`dark/woocommerce/woocommerce-cart-checkout-override.css`

يتم تحميله تلقائيًا على صفحات السلة والدفع عند استخدام dark-theme.
