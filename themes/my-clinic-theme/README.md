# My Clinic WordPress Theme

ثيم WordPress لمنصة حجز المواعيد الطبية - تصميم عصري ومتجاوب باللغة العربية (RTL)

## المميزات

- ✅ تصميم متجاوب بالكامل (Responsive)
- ✅ دعم RTL (من اليمين لليسار)
- ✅ تصميم عصري وجذاب
- ✅ سهل التخصيص
- ✅ متوافق مع معايير WordPress

## التثبيت

1. قم برفع مجلد الثيم إلى `wp-content/themes/`
2. اذهب إلى لوحة تحكم WordPress > المظهر > الثيمات
3. قم بتفعيل ثيم "My Clinic"

## البنية

```
my-clinic-theme/
├── style.css          # ملف الثيم الرئيسي (مطلوب)
├── functions.php      # وظائف الثيم
├── header.php         # رأس الصفحة
├── footer.php         # تذييل الصفحة
├── index.php          # قالب الصفحة الرئيسية
├── assets/
│   ├── css/
│   │   ├── base/      # ملفات CSS الأساسية
│   │   └── components/ # مكونات CSS
│   ├── js/            # ملفات JavaScript
│   └── images/        # الصور والأيقونات
└── README.md
```

## الصفحة الرئيسية

الصفحة الرئيسية (`index.php`) تحتوي على:

- قسم Hero مع صور متعددة
- نموذج البحث (التخصص، المدينة، اسم الطبيب)
- قسم "لماذا MY CLINIC"
- قسم التخصصات الطبية
- قسم الأطباء الأكثر اختيارًا
- بانر دعائي
- قسم آراء العملاء

## التخصيص

يمكنك تخصيص الثيم من خلال:

1. **تعديل الألوان**: في `assets/css/base/tokens.css`
2. **تعديل الخطوط**: في `functions.php` (Google Fonts)
3. **إضافة قوائم**: من لوحة التحكم > المظهر > القوائم

## المتطلبات

- WordPress 5.0 أو أحدث
- PHP 7.4 أو أحدث
